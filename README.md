# ETL-Movies-DW
ETL-Movie-DW is an end-to-end ETL pipeline designed to harmonize and structure data from two complex public movie datasets: IMDB Top Movies and TMDB Enrichment. Python is utilized for crucial initial data pre-processing and extraction tasks, ensuring data quality before staging. The main transformation and loading process is managed within SQL Server (MSSQL) using stored procedures to apply robust data cleansing, integration, and loading into a Star Schema Data Warehouse. The final structure provides a unified, analysis-ready platform for deep insights into global film trends and profitability.
